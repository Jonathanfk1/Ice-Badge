#### Building
The command to compile the project using linux `javac` terminal tool:

`javac -cp '.:server/ngnrtFramework.jar' src/*/*.java src/main.java -d bin/ -encoding iso-8859-1`


#### Executing
 To execute it:
1. 	First run `cd bin` to navigate inside bin/ folder
2. 	Then, `java main`.
